dotfiles
====
My dotfiles


Environments
----
- OS X El Capitan 10.11


Setup
----
```console
$ make brew
$ make shell       # after installing zsh
$ make symlink     # after syncing ~/Dropbox/.share
```


Files in this repository
----
| directory | files |
| --- | ----- |
| init | init scripts |
| linked | dotfiles linked to $HOME |
| spec | serverspec tests for localhost |
